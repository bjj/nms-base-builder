<a name="key-features"></a>
## Key Features

* Build bases for No Mans Sky using proxy representations for building parts.
* Complete freedome of movement when placing building parts - want something upside down? Up in the air? You can place it that way!
* Define "presets" of building parts.
* Save and Load bases.
* Import and Export base data compatable with the No Mans Sky Save Editor.


<br />

<a name="requirements"></a>
## Requirements
* No Mans Sky Save Editor (https://nomansskymods.com/mods/no-mans-sky-save-editor/)
* Autodesk Maya
<br />

<a name="how-to-use"></a>

## Installation and How To Use

1. Download or clone this repository.
2. Extract the contents of the "src" folder to your Documents/maya/scripts folder.
3. Run the following as a python script in the Maya script editor. You can also assign this as a hotkey or shelf button in Maya.

Maya

```python
import nms_builder.ui
nms_builder.ui.load()
```
